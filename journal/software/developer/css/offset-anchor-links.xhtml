Offset Anchor Links

Anchor links allow users of a webpage to jump to specific sections within. This convenience is often exposed through a table of contents.

Unfortunately, if your webpage has a navigation bar, anchor links may appear underneath it by default. To offset anchor links to account for the height of the navigation bar, you may find the discussion in the comments of this blog post beneficial. Specifically, you should take note of this comment:

:target:before {
content: "";
display: block;
height: 30px;
margin: -30px 0 0;
}
